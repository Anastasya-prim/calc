# Калькулятор стоимости корпусной мебели (V8)

Калькулятор для расчёта стоимости корпусной мебели по размерам, материалу и доп. опциям.

## Запуск

Откройте `index.html` в браузере (двойной клик по файлу).

## Формула

`базовая_цена = (ширина × высота × глубина / 1000) × коэффициент_материала`  
`итог = базовая_цена + сумма_выбранных_опций`

## Материалы

| Материал | Цена за дм³ |
|----------|-------------|
| ДСП      | 150 ₽       |
| МДФ      | 250 ₽       |
| Массив   | 500 ₽       |

## Доп. опции

| Опция              | Цена    |
|--------------------|---------|
| Стеклянная дверца  | 5 000 ₽ |
| Обувница           | 2 000 ₽ |
| Подсветка          | 3 000 ₽ |

## V7: Итоговая сводка

Блок «Итоговая сводка (детализация расчёта)» справа от формы — разбивка по материалу, опциям и итогу.

## V8: Сохранение расчётов

В блоке сводки: поле «Наименование расчёта» (обязательное) и кнопка «Сохранить текущий расчёт». Справа — блок «Сохранённые расчёты»: список с названием, датой/временем, параметрами и итогом; кнопки «Загрузить» и «Удалить». Данные хранятся в localStorage.
