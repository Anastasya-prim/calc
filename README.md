# Калькулятор стоимости корпусной мебели (V2)

Калькулятор для расчёта стоимости корпусной мебели по размерам и материалу.

## Запуск

Откройте `index.html` в браузере (двойной клик по файлу).

## Формула

`(ширина × высота × глубина / 1000) × коэффициент_материала = цена в рублях`

## Материалы

| Материал | Цена за дм³ |
|----------|-------------|
| ДСП      | 150 ₽       |
| МДФ      | 250 ₽       |
| Массив   | 500 ₽       |
